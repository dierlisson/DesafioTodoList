# Desafio Final - Lista de Tarefas (To-Do List)

Este projeto é uma simples lista de tarefas, desenvolvido como desafio final da trilha Mobile do curso Norte Tech da Escola Nova Era Tech, proposto pelo professor Pedro Moura.

## Tecnologias Utilizadas

*   **Compose:** A interface do aplicativo foi construída utilizando o Jetpack Compose, o moderno kit de ferramentas de UI do Android.
*   **Kotlin:** A linguagem de programação utilizada foi o Kotlin.

## Armazenamento de Dados

As tarefas são armazenadas de maneira simples em uma `data class` chamada `Tarefa`.
